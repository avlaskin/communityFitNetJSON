# CommunityFitNet - JSON data

This repository contains the JSON data for the CommunityFitNet project.
The CommunityFitNet project has been written in 2019-2020 and Python and frameworks
do not allow to read the provided pickle data at ease anymore.

To run conversion code you need to have:

-  Python 3.9 installed
-  Pandas 1.1.3 library installed
-  NumPy 1.24.4 library installed
